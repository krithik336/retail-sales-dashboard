# Retail Sales Analytics Dashboard

An interactive Business Intelligence and Machine Learning dashboard built using Python and Streamlit.

This project performs end-to-end retail sales analysis, forecasting, visualization, and predictive analytics using real-world retail sales data.

This project focuses on retail business analytics using:

# Forecasting System

The dashboard includes a sales forecasting module using moving average trend analysis.

Forecasting helps businesses:
- analyze future sales trends
- estimate upcoming demand
- identify seasonal patterns
- support business decision-making

---

# Prediction Features

The model predicts future sales using:
- Quantity
- Discount
- Profit

# Downloadable Reports

The dashboard allows users to download machine learning prediction reports directly from the application.

## Included Report Features

- Actual vs Predicted Sales
- Prediction Performance Analysis
- Model Output Results
- Forecasting Summary
- Sales Prediction Report

Users can generate and download reports in CSV format for further analysis and business reporting purposes.
